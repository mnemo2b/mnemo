# Editing

Current workflow in Lightroom Classic.

## Ingest

- Import via Lightroom, copy to external SSD, rename by date + event
- Apply a baseline develop preset (lens correction, mild sharpening)
- Rate quickly: pick (1), reject (X), ignore (no rating)
- Second pass: star ratings on picks (3-5 stars)

## Develop

Order matters:

1. **Crop + straighten** — get the composition final before pixel-level edits
2. **White balance** — eyedropper on a neutral, then taste
3. **Exposure + contrast** — histogram as a guide
4. **Highlights / shadows** — recover detail without flattening
5. **Whites / blacks** — set the endpoints for contrast
6. **Clarity / texture** — use sparingly
7. **Color grading / HSL** — targeted color work
8. **Local adjustments** — masks for sky, subject, dodge/burn
9. **Sharpening + noise reduction** — last, and conservatively

## Heuristics

- Don't chase shadow detail that wasn't there (noise will come for you)
- A small move + walk away + return beats ten large moves
- Watch the edges — straightening often hides unintended crops
- Compare to a reference image or reference print

## Export

- **Web** — sRGB, 2048px long edge, ~80% JPEG quality
- **Print** — sRGB or AdobeRGB depending on lab, full resolution TIFF or high-quality JPEG
- **Social** — short edge 1350px, sRGB, added output sharpening

## Backups

- Lightroom catalog + originals → external drive
- External drive → cloud (Backblaze)
- Critical albums also → second physical drive in a different location
