# Woodworking Basics

Just started. One note, which is honest about where I'm at.

## First projects

Small things that teach technique without needing a shop full of tools:
- Cutting board (flat stock, end grain optional)
- Small box with a lid
- Shaker peg rail
- Simple stool

## Tools I have

- Handsaw (Japanese pull saw — forgiving for beginners)
- Block plane
- Low-angle jack plane
- Chisels (1/4", 1/2", 3/4", 1")
- Combination square
- Mallet
- Clamps (never enough)

## Tools I don't yet have

- Table saw / track saw
- Band saw
- Router
- Jointer / planer

Hand-tool-only is slower but teaches fundamentals. Power tools can come later.

## What I'm learning

- Sharp tools are safer and more pleasant
- Wood moves — grain direction matters
- Cut to a line, don't eyeball
- Square-ness compounds; check at each stage

## Resources

- Paul Sellers' YouTube — hand tools, beginner-friendly
- Rex Krueger — budget and beginner
- *The Anarchist's Tool Chest* — Schwarz, philosophy of minimalism

More here as I build things worth noting.
